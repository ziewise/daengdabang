# Bundle Showroom And Hover Production Prompts

These prompts are for creating bundle campaign assets. Keep every output premium, commercial, and consistent with DaengDaBang's warm storybook-commerce tone.

Target structure:

```text
public/images/bundles/{bundle-slug}/hover.mp4
public/images/bundles/{bundle-slug}/poster.png
public/images/bundles/{bundle-slug}/showroom-1.png
public/images/bundles/{bundle-slug}/showroom-2.png
public/images/bundles/{bundle-slug}/showroom-3.png
```

## Shared Hover Video Rules

Use Flow image-to-video or video generation.

```text
Create a 5-second seamless loop for a premium pet commerce bundle card hover.
Show one dog naturally wearing or using all core items in the bundle at once.
The dog should look comfortable, realistic, and not overloaded.
Use warm natural light, gentle motion, and a polished showroom-ad feel.
No text, no brand logo overlays, no watermark, no extra people, no distorted products.
Camera stays stable. The loop should work as a quiet hover preview.
16:9 landscape, product and dog clearly visible, clean composition.
```

## Shared Showroom Image Rules

Use image generation.

```text
Create a premium pet showroom image for a bundle detail page.
Show a dog naturally wearing or using the selected bundle items.
The scene should feel like a curated ecommerce lookbook, not a busy ad.
Keep products clear and plausible, with no warped straps, floating objects, or impossible anatomy.
No text, no watermark, no logos added, no extra people.
High quality, soft natural light, clean background, realistic dog pose.
Landscape 4:3 or 16:10 composition.
```
