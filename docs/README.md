## 신규 문서 작성 규칙

- 파일명: `숫자-주제.md` (정렬 가능하도록 두자리 prefix)
- 각 문서 최상단에 요약 (출처·일자·핵심 결론)
- 데이터·근거는 표·인용으로 명시
- 결정이 바뀌면 새 문서 추가 (기존 덮어쓰지 않음, 이력 보존)
