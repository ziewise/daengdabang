# 메뉴 구조 검토 — 현재 사이트 vs 신규 메뉴

> **검토일**: 2026-05-12
> **현재 사이트 메뉴**: smartstore.naver.com/daengdabang (사용자 공유)
> **신규 메뉴 정의**: [메뉴 구조 및 열림 방식 정리안] 기준

이 문서는 사용자가 공유한 **현재 운영 중인 스마트스토어 메뉴**와 **신규로 정의한 메뉴 구조**를 비교 검토하여, 보강이 필요한 부분·바꿔야 할 부분을 정리한다.

## 1. 현재 사이트 메뉴 (스마트스토어)

```
베스트
26년 신상
브랜드별 (러프웨어·렉스스펙스·원티그리스·펌블펫솝·카나간·네이처다이어트 등 다수)
러프웨어 (24년 신상 / 25년 신상 / 26년 신상)
렉스스펙스
용품별 (목줄/리드줄, 하네스, 유모차/카시트, 백팩, 의류, 파우치)
사료간식 (사료: 요라/스키너즈/네이처다이어트/카나간/빌리앤마것, 간식: 폴카독)
쿠션식기 (식기, 쿠션/방석)
댕스크림
미용위생 (미용, 위생)
안전장구
장난감류
야옹야옹 (고양이)
전체상품
공지사항
```

### 문제점

| # | 문제 | 영향 |
|---|---|---|
| 1 | 최상위 메뉴가 15개 이상 | 사용자가 한눈에 못 잡음 |
| 2 | **러프웨어·렉스스펙스가 "브랜드별" 안에 + 별도 최상위 메뉴** (중복) | 정보 구조 혼란 |
| 3 | 브랜드명·카테고리·기획성 메뉴가 같은 깊이 | 위계 없음 |
| 4 | "댕스크림" 단일 라인이 최상위 메뉴 | 자체 브랜드 푸시인 듯 |
| 5 | 24/25/26년 신상 따로 (러프웨어 안) | 시간 지나면 오래된 메뉴 잔류 |
| 6 | "야옹야옹" 분리 = 고양이 상품 비중 작음을 인정 | 메뉴 비효율 |
| 7 | "공지사항"이 최상위 헤더에 | 일반적으로 푸터 위치 |

## 2. 신규 메뉴 (우리가 정의한 안)

```
[베스트] [신상품] [카테고리▾] [브랜드▾] [기획전▾]

카테고리 (메가 메뉴):
├ 산책/아웃도어 — 하네스/리드줄, 의류/보호장비, 고글/안전용품
├ 먹거리       — 사료, 간식, 영양/보조간식
├ 생활용품     — 쿠션/침구, 식기/급식용품, 이동가방/외출용품
└ 케어         — 케어/크림, 발바닥 케어, 위생용품

브랜드 (카드형 드롭다운):
├ 러프웨어 (Featured)
├ 렉스스펙스 (Featured)
└ 전체 브랜드 보기

기획전 (상황별 카드):
├ 활동견 추천
├ 산책 필수템
├ 눈 보호 아이템
├ 계절별 추천
└ 할인/이벤트
```

## 3. 비교 매트릭스

| 현재 메뉴 | 신규 매핑 | 상태 |
|---|---|---|
| 베스트 | 베스트 | ✓ 동일 |
| 26년 신상 | 신상품 | ✓ 이름 더 유연 |
| 브랜드별 | 브랜드 > 전체 브랜드 보기 | ✓ 통합 |
| 러프웨어 (단독) | 브랜드 > 러프웨어 (Featured) | ✓ 중복 제거 |
| 렉스스펙스 (단독) | 브랜드 > 렉스스펙스 (Featured) | ✓ 중복 제거 |
| 용품별 | 카테고리 > 산책/아웃도어 + 일부는 생활용품 | ✓ 통합 |
| 사료간식 | 카테고리 > 먹거리 | ✓ 통합 |
| 쿠션식기 | 카테고리 > 생활용품 (쿠션/식기) | ✓ 통합 |
| 미용위생 | 카테고리 > 케어 | ✓ 통합 |
| 안전장구 | 카테고리 > 산책/아웃도어 > 고글/안전용품 | ✓ 통합 |
| **장난감류** | **없음** | ❌ **누락 — 추가 필요** |
| 야옹야옹 | 없음 | ⚠️ **고양이 처리 결정 필요** |
| 댕스크림 | 없음 | ⚠️ **자체 브랜드 노출 결정 필요** |
| 전체상품 | 카테고리 메가메뉴 > "전체 보기" 링크 | + 추가 권장 |
| 공지사항 | 푸터로 이동 | + 푸터 작업 시 반영 |

## 5. 권장 변경 사항

### A. 카테고리 보강 — 장난감/놀이 추가

```
카테고리 (수정):
├ 산책/아웃도어
│  ├ 하네스
│  ├ 리드줄/목줄
│  ├ 의류/보호장비
│  ├ 고글/안전용품
│  └ 이동가방/유모차/카시트
├ 먹거리
│  ├ 사료
│  ├ 간식
│  ├ 영양/보조
│  └ 디저트/음료  ← 댕스크림 컬렉션 흡수 (14개)
├ 생활용품
│  ├ 쿠션/침구
│  └ 식기/급식용품
├ 장난감/놀이   ← 신규 그룹
│  ├ 노즈워크/지능
│  ├ 원반/터그
│  └ 라텍스/봉제 장난감
└ 케어
   ├ 스킨/크림 케어
   ├ 발바닥 케어
   ├ 미용용품
   └ 위생/배변용품
```

> **메가 메뉴 컬럼이 4 → 5로 늘어남**. CSS 그리드를 `repeat(5, 1fr)` 로 변경 필요.
> 대안: 장난감/놀이를 "생활용품" 안의 한 서브항목으로 두기 (4그룹 유지).

### B. 고양이 처리 — 3가지 옵션

| 옵션 | 설명 | 권장도 |
|---|---|---|
| **A. 강아지 전용** | 고양이 상품 11개 제외, 댕다방 브랜드를 강아지에 집중 | ★★★ 권장 (초기) |
| B. 헤더 토글 | `[강아지/고양이]` 상단 토글로 전체 카테고리 분리 | ★★ (고양이 확장 시) |
| C. 카테고리 내 통합 | "고양이 라인업" 그룹 추가 | ★ (관리 복잡) |

→ **A 추천** — 현재 비중 3% (11/345)이고, 향후 고양이 라인업 확장 시 B로 전환 가능

### C. 댕스크림 — "특수 디저트/음료" 큐레이션 메뉴 (정정)

> **2026-05-12 사용자 스크린샷 확인 결과**: 댕스크림은 단일 브랜드 메뉴가 아니라
> **14개 상품의 디저트·음료·노벨티 푸드 큐레이션**이었음. Excel 분석에서 댕스크림 브랜드를
> 1개로 본 것은 오해.

#### 실제 댕스크림 메뉴 구성 (14개)

| 종류 | 상품 |
|---|---|
| 🍦 아이스크림 (5) | 휴고앤셀리 연어맛 / 댕스크림 딸기 / 스윗펫 딸기 / 짜요 100ml·50ml 블루베리 / 리얼치즈 포켓 |
| 🥛 음료·유제품 (4) | 탐라이프 산양유 퍼피·어덜트 / 무지개 야채 스무디 / (베지믹스도 음료성) |
| 🍷 노벨티 주류 (2) | 애나펫 댕와인 / 댕소주 |
| 🍓 요거트·보조 (2) | 프로지스 요거트 / 미티본 베지믹스 100g·475g |
| 기타 | 강아지 수제 |

용도세분류 분포: `간식/트릿` 12개 + `기타/확인필요` 2개 — Excel 분류로는 단순 간식에 묻혀있음.

#### 권장 처리 — 카테고리 + 기획전 동시 노출

**1단계 — 카테고리 안 별도 그룹**
```
먹거리:
├ 사료 (44)
├ 간식 (트릿 + 덴탈 = 41)
├ 영양/보조 (7)
└ 디저트/음료 (14)  ← 신규
   ├ 아이스크림
   ├ 음료/유제품
   └ 특수 (노벨티 주류 등)
```

→ 사용자가 탐색할 때 "강아지 아이스크림 찾고 싶다" 같은 의도에 대응

**2단계 — 시즌별 기획전 카드**

여름 시즌(6~9월): 기획전 5번 카드를 **"댕스크림 컬렉션"** 또는 **"여름 한정 차가운 간식"** 으로 노출.
겨울/평시: 다른 기획전으로 교체 (시즌 컬렉션, 보온 의류 등).

→ 마케팅 동선과 탐색 동선 모두 커버, 댕다방 차별화 포인트 살림

### D. 헤더 / 푸터 분리

| 항목 | 현재 위치 | 권장 위치 |
|---|---|---|
| 공지사항 | 헤더 | 푸터 |
| 고객센터 | (없음) | 푸터 |
| 전체상품 | 헤더 | 카테고리 메가메뉴 하단 "전체 보기" 링크 |

### E. 카테고리 메가메뉴 — "전체 보기" 링크

각 그룹 하단에 `→ 전체 보기` 링크 추가:
```
산책/아웃도어
├ 하네스
├ 리드줄/목줄
├ ...
└ → 전체 보기  (그룹 페이지로 이동)
```

이미 그룹 헤더(h4)가 클릭 가능하게 되어 있지만, "전체 보기" 명시 링크를 두면 발견성 ↑

## 6. 브랜드 메뉴 — 데이터 기반 보강 검토

현재 브랜드 메뉴는 러프웨어·렉스스펙스만 Featured. 하지만 Excel을 보면 다음 브랜드도 분량이 작지 않음:

| 순위 | 브랜드 | 상품 수 | 비중 |
|---:|---|---:|---:|
| 3 | 요라 | 24 | 7% |
| 4 | 아페토 | 18 | 5% |
| 5 | 수파 | 18 | 5% |
| 6 | 이비야야 | 16 | 5% |
| 7 | 주스누드 | 15 | 4% |

**옵션**: 브랜드 드롭다운 하단에 "인기 브랜드" 칩 리스트 추가
```
브랜드 드롭다운:
├ [Ruffwear 큰 카드] [Rex Specs 큰 카드]
├ 인기 브랜드
│  요라  아페토  수파  이비야야  주스누드  카나간 ...
└ → 전체 브랜드 보기
```

또는 그대로 두고 "전체 브랜드 보기" 페이지에서 모든 브랜드 노출. **권장: 1단계는 그대로**, 사용자 분석 후 인기 브랜드 노출 결정.

## 7. 최종 권장 메뉴 구조

```
헤더:
  [로고] [베스트] [신상품] [카테고리▾] [브랜드▾] [기획전▾]   [🔍] [🛒] [로그인]

카테고리 (메가 메뉴):
├ 산책/아웃도어     하네스 / 리드줄·목줄 / 의류·보호장비 / 고글·안전용품 / 이동가방·유모차
├ 먹거리           사료 / 간식 / 영양·보조 / 디저트·음료 (구 댕스크림)
├ 생활용품         쿠션·침구 / 식기·급식용품
├ 장난감/놀이      노즈워크 / 원반·터그 / 라텍스·봉제      ← 신규
└ 케어             스킨·크림 / 발바닥 / 미용 / 위생·배변

브랜드 (카드형):
├ Ruffwear (Featured)
├ Rex Specs (Featured)
└ → 전체 브랜드 보기 (35개)

기획전 (상황별):
├ 활동견 셀렉션
├ 장마·우천 필수템  (또는 계절별 추천)
├ 눈·청력 보호
├ 프리미엄 푸드
└ 시즌 컬렉션

푸터:
├ 공지사항 / 고객센터 / 이용약관 / 개인정보처리방침
├ 사업자 정보
└ SNS·뉴스레터
```

## 9. 변경 시 영향 받는 파일

장난감/놀이 추가 등 메뉴 구조 변경 시 수정 대상:

| 파일 | 수정 위치 |
|---|---|
| `assets/css/header.css` | `.mega-menu-inner { grid-template-columns: repeat(4, 1fr) → repeat(5, 1fr) }` |
| `main.html` | 카테고리 메가 메뉴 HTML (4그룹 → 5그룹) |
| `main.html` | 모바일 슬라이드 패널의 아코디언 (장난감 그룹 추가) |
| `menu-preview.html` | 동일하게 반영 |
