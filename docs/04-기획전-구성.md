# 기획전 구성 — 5개 테마

> **작성일**: 2026-05-12
> **기준 데이터**: 카탈로그 345개 → 고양이 12개 제외 = **333개**
> **적용 위치**: 헤더 드롭다운 + 메인 페이지 기획전 섹션

이 문서는 댕다방 쇼핑몰의 **상황별 기획전** 5개 테마가 어떤 기준으로 구성되었는지, 어떤 상품들이 들어가는지를 정리한다.

---

## 1. 구성 원칙

기획전은 단순히 카테고리를 나열하는 게 아니라, **사용자의 상황·목적·시즌**에 맞춰 큐레이션 동선을 제공한다. 그래서 다음 원칙으로 5개를 정했다.

1. **5개 다른 동선 커버** — 활동 / 시즌 / 차별화 / 먹거리 / 동적 마케팅
2. **각 테마는 실제 카탈로그 분량 보유** (분량 없는 테마 = 빈 슬롯 만들지 않음)
3. **중복 동선 제거** — 활동견과 산책 필수템처럼 겹치는 건 통합
4. **1개 슬롯은 동적**으로 시기별 노출 변경 (시즌 마케팅)
5. **Featured(메인 큰 카드) 1개 + 일반 카드 4개** 구성

## 5. 운영 가이드

### 시즌 슬롯 전환 시점
- 매월 1일 운영자가 시즌에 맞춰 5번 슬롯 데이터를 교체
- 추후 자동화 가능 (날짜 기반 동적 노출)

### 카드 노출 우선순위
1. **Featured (활동견)** — 항상 좌측 큰 카드, 변경 없음
2. **장마·우천 / 눈·청력 보호 / 프리미엄 푸드** — 우측 4개 카드 중 3개 고정
3. **시즌 슬롯** — 우측 4개 카드 중 1개, 시기별 동적

### 측정 지표 (추후 운영 시)
- 각 카드 클릭률
- 카드 → 상품 클릭률
- 카드 → 구매 전환율
- 시즌별 5번 슬롯 효과 (전환율 변화)
