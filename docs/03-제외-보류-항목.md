- 댕다방의 메인 브랜드 포지셔닝은 **강아지(댕댕이)** 중심
- 고양이 상품은 카탈로그 345개 중 **12개 (3.5%)** — 비중 작음

- 카테고리 안에 섞으면 정보 구조 혼란

→ **현재는 강아지 전용으로 출시**, 향후 고양이 라인업 확장 시 재검토

### 강아지·고양이 공용 상품 — 별도 검토 가치

다음 2개는 **상품 자체가 강아지·고양이 공용**이라 제외하지 않고 가져갈 수 있음:

- `프로지스 요거트 블루베리 강아지 고양이 4개입` — 강아지에게도 판매 가능
- `탐라이프 산양유 퍼피&키튼 200ml` — 강아지 새끼(퍼피)도 대상

→ **권장**: 이 2개는 상품명에서 "고양이/키튼" 부분만 표시 조정하거나, 그대로 강아지 카테고리에 포함. 추후 결정 필요.

### 데이터 추적

향후 다시 들여올 때 필요한 정보:
- **Excel 시트**: `전체_세분류`
- **검색 키워드**: `고양이`, `캣`, `퍼피&키튼`, `&캣`
- **No 컬럼 값**: 위 표의 12개 No
